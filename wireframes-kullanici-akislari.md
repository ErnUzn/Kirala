# Kirala - Wireframes ve Kullanıcı Akışları

## 1. Ana Kullanıcı Akışları

### Kiralama Yapan Kullanıcı (Kiracı) Akışı
1. Ana Sayfa → Kategori Seçimi → Ürün Listeleme → Ürün Detay → Tarih Seçimi → Ödeme → Onay
2. Ana Sayfa → Arama → Ürün Listeleme → Ürün Detay → Tarih Seçimi → Ödeme → Onay
3. Profil → Kiralamalarım → Kiralama Detayı → Değerlendirme

### Ürün Kiralatan Kullanıcı (Kiraya Veren) Akışı
1. Profil → Ürünlerim → Yeni Ürün Ekle → Ürün Detayları → Fotoğraf Ekle → Fiyatlandırma → Yayınla
2. Ana Sayfa → Kendi Ürünüm → Düzenle → Güncelle
3. Profil → Kiralama Talepleri → Talebi Onayla/Reddet → Mesajlaşma

### Admin Akışı
1. Admin Giriş → Dashboard → Ürün Yönetimi → Ürün Ekle/Düzenle/Sil
2. Admin Giriş → Dashboard → Kullanıcı Yönetimi → Kullanıcı Detayları
3. Admin Giriş → Dashboard → Kiralama İşlemleri → Kiralama Detayları
